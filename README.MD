## :information_source: Description

This is an ecommerce API designed to be useful for stores owners, as it allows some message automation among other tasks, like getting order information from Aliexpress, Correios and Shopify.

<div align="center" style="text-align: center;">
  <img src="https://i.giphy.com/media/iEL7eGgihmtCs9cBsl/giphy.webp">
</div>

## :information_source: Features

- [x] Whatsapp bot automation routes;
- [x] Telegram bot automation routes;
- [x] Get order information from Aliexpress;
- [x] Get order information from Shopify;
- [x] Get order information from Brazil post office, called Correios.

## :information_source: How to edit it

To clone this repository and make changes in the source code, you'll need [Git](https://git-scm.com) and  [Nodejs](https://nodejs.org/en/) in your computer.

```bash
# Clone this repository
$ git clone https://github.com/lucasvtiradentes/ecommerce-stores-api

# Go into the repository folder
$ cd ecommerce-stores-api

# Install dependencies
$ npm install

# Run the code in devlopment mode
$ npm run dev
```

To build the production code, run the following code:

```bash
# Clone this repository
$ npm run build

# Run the code in production mode
$ npm run start
```
